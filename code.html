The Code for Taco Cat.

                        
                            function getValues(){
                                document.getElementById("resultArea").classList.add("invisible");
                                let userString = document.getElementById("userString").value;
                            
                                if(userString == ""){
                                    alert("Please enter a valid string")
                                }
                                else{
                                let newString = userString.toLowerCase().replace(/[^a-z0-9]/g,'');
                                let revString = reverseString(newString);
                                displayResults(newString, userString, revString);
                                }
                            }
                            
                            function reverseString(newString){
                                let revString = [];
                                for (let index = newString.length-1; index >= 0; index--) {
                                    revString += newString[index];        
                                }
                                return revString;
                            }
                            
                            function displayResults(newString, userString, revString){
                            
                                if(newString == revString){
                                    document.getElementById("displayResults").innerHTML = `Success!! ${userString} is a palindrome!`
                                }else { document.getElementById("displayResults").innerHTML = `${userString} is not a palindrome.`}
                                document.getElementById("resultArea").classList.remove("invisible");
                            }

The code is structured in three main functions.

getValues()

The first function initially makes sure that the area we will display the results in is invisible. Next, it gets the string the user entered and checks to make sure it's not empty.

The string is set to a new variable while setting all characters to lower case and removing special characters and white spaces. The reason for declaring a new variable is so the original string can be passed to the last function where it can be displayed back to the user.

The new string is then passed to the second function...

reverseString()

The second function declares an empty array and then populates the array with the string backwards by using a 'for loop' to iterate through it starting at the last index.

displayResults()

The final function displays the results based on a comparison between the reversed and non-reversed string. The original string entered by the user is displayed in the result in red. The last step is to remove the 'invisible' class so the results are visible.
